# intrepion-saying-hello-json-rpc-client-web-typescript-react-typescript

## Commands

### Install

```bash
npm install
```

### Test

```bash
npm test
```

### Initialize Database

```bash
./scripts/init_postgres.sh
```

### Run

```bash
REACT_APP_SERVER_URL="http://localhost:5220" npm start
```
